6.2.3.1 AISC_EM11_Pset_Weld

Weld:

Weld property set is assigned to an instance of Weld. Weld shall use Fastener (IfcFastener) for representing location and use Connects With Realizing Elements (IfcRelConnectWsithRealizingElements) for relation between connecting piece. Weld for rolled plate pipe cannot use Connects With Realizing Elements as it violates IFC schema definition. Therefore, it should use Connects Elements (IfcRelConnectsElements) and set IfcFastener as Related Element. Geometric Representation of welding should be curve that shows welding path.

Weld Size of weld. Physical extents and dimensions of weld. Weld is modeled and shown as a theoretical physical object using throat/size etc so that a weight and estimate can be produced from the model.
