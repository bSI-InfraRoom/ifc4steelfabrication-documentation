IfcCurveWeightsPositive

Semantic definitions at the function
Function Definition

This function checks whether the curve weights of a b-spline are positive.

NOTE  Corresponding STEP function: curve_weights_positive. Please refer to ISO/IS 10303-42:1994, p.115 for the final definition of the formal standard. 

HISTORY  New function in IFC Release 2x Edition 2.

Formal representations
EXPRESS Specification

FUNCTION IfcCurveWeightsPositive
(B: IfcRationalBezierCurve) : BOOLEAN;
   LOCAL
     Result : BOOLEAN := TRUE;
   END_LOCAL;

   REPEAT i := 0 TO B.UpperIndexOnControlPoints;
     IF B.Weights[i] <= 0.0  THEN
       Result := FALSE;
       RETURN(Result);
     END_IF;
   END_REPEAT;
   RETURN(Result);
END_FUNCTION;
