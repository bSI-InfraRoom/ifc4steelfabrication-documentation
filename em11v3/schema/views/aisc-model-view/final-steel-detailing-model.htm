1.1.1 Final Steel Detailing Model

ProcessSenderReceiver
31-60 00 00 Implementation Phase34-20 11 21 Engineer34-35 10 14 Fabricator

The function of the model is to provide enough detailed information about the steel members to enable the fabricator to manufacture the steel pieces. The model is the finalized version of "Steel Detailing Model" which is provided as the output any modification and integration of final detailing model. During the steel fabrication the plant management software will add member status to the model.

The exchange model includes all the finalized information of 'Advanced Steel Detailing Model' like the member ID, and member properties (section size, grade, length) detailed welding and bolting information, information about holes and member camber, etc. for all main and minor members. The model also includes detailed design of all required connections including safety connections.

Optional information: erection sequences and erection views, 4D simulated schedule. 4. Example software used: Revit Structure, SmartPlant3D, FrameWorks-Plus, SDS/2, Tekla, StruCAD.

Passing information from detailing software to shop floor for automated fabrication.
