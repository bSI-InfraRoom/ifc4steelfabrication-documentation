5.2.4.8 AISC_EM11_Pset_Castellation

Castellation: Castellation has a repeating hole feature along its length. The repeating feature may be given by its type. It may also define spacing, height and width.

buildingSMART Data Dictionary

PSD-XML

NameTypeDescription
CastellationType P_ENUMERATEDVALUE / AISC_EM11_CastellationType
Castellation TypeCastellation Type declares the classification of the type of longitudinal repeating feature by its shape. Value can be one of :
  • Circular
  • Hexagonal
  • Octagonal
  • Undefined
CastellationEndPostWidth1 P_SINGLEVALUE / IfcPositiveLengthMeasure
Castellation End Post Width 1Castellation End Post Width 1 in IfcLengthMeasure type, provides the numerical value with an appropriate unit of the linear dimension to the first web opening measured from the first cut end.
CastellationEndPostWidth2 P_SINGLEVALUE / IfcPositiveLengthMeasure
Castellation End Post Width 2Castellation End Post Width 2 in IfcLengthMeasure type, provides the numerical value with an appropriate unit of the linear dimension to the last web opening measured from the last cut end.
CastellationSpacing P_SINGLEVALUE / IfcPositiveLengthMeasure
Castellation SpacingCastellation Spacing in IfcLengthMeasure type, specifis the numerical value with an appropriate unit of the linear distance between each feature, measured along the length of the part.
CastellationHeight P_SINGLEVALUE / IfcPositiveLengthMeasure
Castellation HeightCastellation Height in IfcLengthMeasure type, specifies the numerical value with an appropriate unit of the linear distance between the top and bottom of each feature, measured across the depth of the part.
CastellationWidth P_SINGLEVALUE / IfcPositiveLengthMeasure
Castellation WidthCastellation Width in IfcLengthMeasure type, specifies the numerical value with an appropriate unit of the linear distance between the extreme edges of each feature, measured along the length of the part.
CastellationDepth P_SINGLEVALUE / IfcPositiveLengthMeasure
Castellation DepthCastellation Depth in IfcLengthMeasure type, specifies the numerical value with an appropriate unit of the linear distance between the top of the part and the extreme edges of each feature, measured down the depth of the part.
