5.2.4.2 AISC_EM11_Pset_AssemblyIdentification

Assembly Identification: An assembly can have identification information. This property set is for an Assembly represented in IfcElementAssembly

buildingSMART Data Dictionary

PSD-XML

NameTypeDescription
AssemblyMark P_SINGLEVALUE / IfcIdentifier
Assembly MarkA short text reference for the assembly. A means of identifying the assembly within the fabrication shop.
ClientMark P_SINGLEVALUE / IfcIdentifier
Client MarkA short text reference that may be used to identify the assembly for the client.
PrelimMark P_SINGLEVALUE / IfcIdentifier
Prelim MarkA short text reference that may be used to identify the assembly at the bidding or tender stages of the assembly’s production life.
ShippingMark P_SINGLEVALUE / IfcIdentifier
Shipping MarkA short text reference that may be used to identify the assembly at the shipping stages of the assembly’s production life.
BarCode P_SINGLEVALUE / IfcIdentifier
BarcodeA numerical interpretaion of the bar code associated with the assembly. The bar code is sometimes used to identify the assembly in the fabrication shop, during transportation, and during erection using a bar code scanner.
