IfcLengthMeasure

Natural language names
Semantic definitions at the type
Type definition

Definition from ISO/CD 10303-41:1992: A length measure is the value of a distance.

Usually measured in millimeters (mm).

Type: REAL

NOTE Corresponding STEP name: length_measure, please refer to ISO/IS 10303-41:1994 for the final definition of the formal standard.
HISTORY New type in IFC Release 1.5.1.
