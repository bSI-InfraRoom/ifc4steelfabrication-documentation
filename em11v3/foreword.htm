Foreword

Industry Foundation Classes (IFC) is a data model for the building construction industry, listed as ISO 16739. It spans various disciplines including architecture, engineering, construction, and operations. This data model is partitioned and specialized into subsets, each referred to as a Model View Definition (MVD). An MVD defines a subset of the IFC schema, defines various information exchanges with requirements for data to be included, and documents use of data structures tailored to a particular discipline.

The AISC BIM Standards define the general requirements for entire life cylce of the structural steel including design, analysis, manufacture, and construction.
